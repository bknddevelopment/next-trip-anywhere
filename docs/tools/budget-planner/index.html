Smart Budget Tips

  • • Set aside 10% for emergencies
  • • Track spending daily on cruise
  • • Use ship's app to monitor charges
  • • Set onboard spending alerts
  • • Prepay gratuities to lock in costs
  • • Book excursions early for discounts

Common Budget Pitfalls

  • • Forgetting gratuities (15-20% extra)
  • • WiFi charges ($20-30/day)
  • • Specialty dining adds up quickly
  • • Casino and bar tabs
  • • Professional photos ($200+)
  • • Spa treatments at sea prices

Payment Planning

  • • Notify banks of travel dates
  • • Bring 2-3 payment methods
  • • Cash needed for port vendors
  • • No foreign transaction fee cards
  • • Set up onboard account carefully
